# <a name="storage-explorer-accessibility"></a>Storage Explorer – kisegítő lehetőségek

## <a name="screen-readers"></a>A képernyőolvasók

Storage Explorer egy képernyőolvasó használatát támogatja a Windows és Mac rendszerre. A következő képernyőolvasók az egyes platformokra vonatkozó ajánlott:

Platform | Képernyőolvasó
---------|--------------
Windows  | NVDA
Mac      | Narrátor
Linux    | (a képernyőolvasók nem támogatottak a Linux)

Ha accessibility hibát amikor a Storage Explorer használatával, kérjük, [nyisson egy problémát a Githubon](https://github.com/Microsoft/AzureStorageExplorer/issues).

## <a name="zoom"></a>Nagyítás

Akkor is nagyíthat a Storage Explorerben nagyítás keresztül. Nagyítás, kattintson a **nagyítás** a Súgó menü. A Súgó menü kicsinyítéshez és alaphelyzetbe állítása a nagyítási szint vissza az alapértelmezett szint is használhatja.

![Nagyítás beállításai a Súgó menü][0]

A Nagyítás beállítása a legtöbb felhasználói felületi elemek méretét is megnöveli. Ajánlott is lehetővé nagy és nagyítás győződjön meg arról, hogy a felhasználói felület minden eleme megfelelően vannak-e méretezve, hogy az operációs rendszer beállításait.

## <a name="high-contrast-themes"></a>Kontrasztos megjelenítésű téma

Storage Explorer rendelkezik két kontrasztos megjelenítésű téma, **Magas kontraszt világos** és **Magas kontraszt sötét**. Válassza ki a segítségével módosíthatja a téma > témák menü.

![Témák almanüje][1]

A téma beállítása a legtöbb felhasználói felületi elemek színének módosítása. Ajánlott a is engedélyezheti az operációs rendszer megfelelő, győződjön meg arról, hogy a felhasználói felület minden eleme megfelelően színűek Kontrasztos megjelenítésű téma.
