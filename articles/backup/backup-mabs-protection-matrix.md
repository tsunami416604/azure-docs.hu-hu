## <a name="cluster-support"></a>Fürtök támogatása
Az Azure Backup Server védheti az adatokat az alábbi fürtözött alkalmazásokban:

-   Fájlkiszolgálók

-   SQL Server

-   Hyper-V - védelme egy Hyper-V fürt kiterjesztett DPM védelemmel, nem adhat másodlagos védelmet a védett Hyper-V munkaterhelésekhez.

    Ha a Hyper-V a Windows Server 2008 R2 rendszeren futtatja, ügyeljen arra, hogy telepítse a Tudásbáziscikkben leírt frissítést [975354](https://support.microsoft.com/en-us/kb/975354).
    Ha Hyper-V a Windows Server 2008 R2 rendszeren, fürtkonfigurációban futtatja, ellenőrizze, hogy telepítette az SP2 és a KB-os [971394](https://support.microsoft.com/en-us/kb/971394).

-   Exchange Server – az Azure Backup Server védheti a nem megosztott lemezfürtöket a támogatott Exchange Server-verziók (fürt folyamatos replikálásával), és a folyamatos helyi replikálásra konfigurált Exchange Servert is védheti.

-   SQL Server – az Azure Backup Server nem támogatja a fürt megosztott köteteinek (CSV) lévő üzemeltetett SQL Server-adatbázisok biztonsági mentése.

Az Azure Backup Server is védeni, amelyek a DPM-kiszolgálóéval azonos tartományban, és a egy gyermek és megbízható tartományokban lévő találhatók. Ha azt szeretné, nem megbízható tartományban vagy munkacsoportban levő adatforrások védelmét, használjon Tanúsítványalapú hitelesítés, illetve egy önálló kiszolgáló esetén NTLM vagy Tanúsítványalapú hitelesítés csak a fürt számára.
