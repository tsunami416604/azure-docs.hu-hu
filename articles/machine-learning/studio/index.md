# <a name="azure-machine-learning-studio-documentation"></a>Az Azure Machine Learning Studio dokumentációja

Az Azure Machine Learning Studio olyan, az együttműködést támogató, egérrel kezelhető eszköz, amellyel az adataihoz prediktív elemzési megoldások hozhatók létre, tesztelhetők és helyezhetők üzembe.
Az oktatóanyagok, videók és példamodellek bemutatják a Machine Learning-modellek felépítését és üzembe helyezését a Studio használatával.
