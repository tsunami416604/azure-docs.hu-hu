# <a name="how-to-sign-up-for-the-translator-speech-api"></a>A Translator Speech API a regisztráció

Kövesse az alábbi lépéseket a Translator Speech API regisztrálhat.

## <a name="1-sign-into-azure"></a>1: Jelentkezzen be az Azure

- Még nincs fiókja? [A Microsoft Azure-fiók](http://azure.com/)
- Van már fiókja? [bejelentkezés](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2: előfizetés a Microsoft Translatorhoz

- Miután bejelentkezett az Azure, nyissa meg a [Azure Cognitive Services](https://portal.azure.com/#create/Microsoft.CognitiveServices) szakasz
- "API"típusa alatt válassza ki a Translator Speech API
- A Tarifacsomag területen válassza ki a tarifacsomagot, amely leginkább megfelel az igényeinek
    - Minden előfizetési csomaghoz tartozik egy ingyenes szintet. Az ingyenes csomag ugyanazokat a szolgáltatásokat és funkciókat, mint a fizetős csomagok rendelkezik, és nem rendelkezik a lejárati dátumot.
    - Csak egy ingyenes előfizetéssel rendelkezhet a fiókjában
- Adja meg a képernyőn a többi, és válassza a Létrehozás gombra
- Az előfizetés azonnali használatra készen áll

## <a name="3-authentication-key"></a>3: hitelesítési kulcs

- A hitelesítési kulcs lekéréséhez lépjen a minden erőforrás menüben > kattintson az előfizetésre
- A kulcs értékét használja a hitelesítéshez, a kulcs-érték bizalmas jellegének megőrzése. Szüksége lesz a kulcs értékét, amikor az alkalmazás fejlesztése

## <a name="4-learn-test-and-get-support"></a>4: ismerje meg, tesztelése és támogatás kérése

- [Hitelesítésikód-példák a Githubon](https://github.com/MicrosoftTranslator)
