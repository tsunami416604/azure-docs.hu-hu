# <a name="develop-secure-applications-on-azure"></a>Az Azure-ban a biztonságos alkalmazások fejlesztése
## <a name="abstract"></a>Absztrakt
Ez a tanulmány a biztonsági kérdéseket és a vezérlőket is mindegyik fázisban, a szoftverek fejlesztési életciklus során megfontolandó a felhőalapú alkalmazások fejlesztésének általános útmutató. Ezek a fogalmak végrehajtási, a termék viselkednek segíthet a biztonságosabb szoftvereket. A jelen dokumentumban ismertetett javaslatok az Azure biztonsági és a élményt ügyfeleinek Tapasztalataink származnak.

Ez a tanulmány az célja, hogy egy erőforrás szoftver tervezők, fejlesztők számára, és a tesztelőknek minden szintjén, hogy hozhat létre és biztonságos Azure-megoldások üzembe helyezése.
