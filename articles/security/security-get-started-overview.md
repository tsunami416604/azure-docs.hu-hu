# <a name="azure-security-overview"></a>Az Azure biztonsági szolgáltatásainak áttekintése

Az Azure.com webhelyen a Microsoft Azure biztonsági információk hely az Azure security műszaki kérdéseire az a hely. Ha érdekli az Azure megfelelőségi, és hogyan védi az Azure a saját infrastruktúráját, keresse fel a [Microsoft Trust Center](https://microsoft.com/en-us/trustcenter/default.aspx).

Tudjuk, hogy biztonsági-e a feladatot a felhőben, és hogy mennyire fontos, hogy látja-e az Azure security pontos és időben információt. A legjobb okok igénybe az Azure az alkalmazások és szolgáltatások egyik biztonsági eszközökkel és a rendelkezésre álló lehetőségek széles választékának előnyeinek kihasználása érdekében. A következő eszközök és képességek segítségével biztonságos megoldásokat hozhat létre a biztonságos Azure-platformon lehetővé teszik, hogy.

Ezen a lapon, és ez a hely fejlődik, online állapotba az új Azure-szolgáltatásokat és a meglévő szolgáltatások frissülnek. Biztonsági kínálatunk folyamatos javítása és fog osztjuk meg ezeket a fejlesztéseket itt technikai részleteit. Azt javasoljuk, hogy rendszeresen térjen vissza erre a helyre, és megtekintheti az új!

## <a name="using-the-azure-security-information-site"></a>Az Azure biztonsági információk hely kapcsolat használatával

Ha az oldal bal oldalán, további biztonsági témakörök kategóriák közül. Kattintson vagy koppintson a témakörben tekintse meg a kategória van a cikkeket.

Mindegyik kategóriában található egy áttekintő cikket. Az áttekintő cikk célja, hogy összefoglalja, az Azure által biztosított lehetőségeket a kategória biztonsági tekintetében. Az áttekintő cikkek segítségével megismerheti a termékek, szolgáltatások és funkciók, amelyek az üzembe helyezést a biztonság növelése érdekében használhatja Azure-ban elérhető. Ezek a cikkek áttekintése műszaki tartalom számos olyan hivatkozásokat tartalmaznak az, hogy a témakörök és azok többet is megtudhat.

Az Azure biztonsági információk hely jelenleg megoldást a következő kategóriákban:

* További források
* Hálózati biztonság
* Tárbiztonság
* Virtuális gépek biztonsága
* Endpoint protection (Végpontok védelme)
* Biztonságkezelés és megfigyelés
* Identitáskezelés
* PaaS szolgáltatások
* Adott iparági
* Architektúra

> [!NOTE]
> Az Azure biztonsági információk helyen található összes cikk az oldal bal oldali navigációs jelennek meg. Ha meg szeretné tekinteni a összes cikk, kattintson vagy koppintson **Továbbiak** alján, a kategóriák listában.
>
