# <a name="azure-cdn-coverage-by-metro"></a>Az Azure CDN lefedettség Metro szerint 
> [!div class="op_single_selector"]
> * [POP-helyek régiónként](cdn-pop-locations.md)
> * [POP-helyek rövidítés alapján](cdn-pop-abbreviations.md)
> 


Ez a cikk felsorolja az aktuális metrót tartalmazó jelenléti pontok (POP) helyen, régió, az Azure Content Delivery Network (CDN) termékek szerint rendezve. Minden egyes Metro több mint egy felugró tartalmazhat. Például a Microsoft Azure CDN 129 POP rendelkezik az 61 metrót között. 

> [!IMPORTANT]
> Város POP-helyek az **Akamai Azure CDN** külön-külön nem kerül.  
> 
> Mivel minden egyes Azure CDN terméket egy különálló lehetőség a hozzá tartozó CDN-infrastruktúrák, a Microsoft javasolja, szemben a POP-helyek használatával dönthet arról, hogy melyik Azure CDN terméket használja. Ehelyett érdemes végfelhasználói teljesítmény és szolgáltatásai. Az egyes Azure CDN termékek kiválasztása a felhasználók számára a megfelelő termék teljesítményének tesztelése.
