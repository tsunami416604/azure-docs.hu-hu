# <a name="saas-fulfillment-apis-version-1-deprecated"></a>SaaS-teljesítési API-k 1-es verziója (elavult)

> [!WARNING]
> A SaaS-teljesítési API ezen kezdeti verziója elavult; Ehelyett használja a [SaaS-beteljesülés API v2](./pc-saas-fulfillment-api-v2.md)-t.  Az API ezen kezdeti verzióját jelenleg csak a meglévő közzétevők kiszolgálására kell fenntartani. 

Az API ezen verziójának dokumentációja már nem érhető el. Ha úgy érzi, hogy hozzá kell férnie, vegye fel a kapcsolatot saas-api-v1@microsoft.com .

Az összes új fejlemény esetében váltson [az API v2-es verziójára](./pc-saas-fulfillment-api-v2.md) , még akkor is, ha már használja a v1-et. Ez a verzió a jövőben teljesen elavulttá válik.
