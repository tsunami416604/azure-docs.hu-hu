<a name="seller-insights-definitions"></a>Értékesítői Insights definíciók
=======================

Az alábbi táblázatban számos, az értékesítői Insights használt kifejezések definícióit.

|  **Kifejezés**                 |  **Definíció**                                                                                                                              |
|  --------------------------------------------    |  ---------------------------------------------------------------------------------------------------------------------------------           |
| **Az Azure licenc típusa**                               | Az ügyfelek használják az Azure megvásárolható licencszerződés típusa.  Más néven a csatornát.                                                  |
| **Azure License Type: Cloud Solution Provider**      | A végfelhasználó ügyfél vásárol az Azure-ban és a Piactéri ajánlat használatával a Cloud Solution Provider, akik viszonteladójával funkcionál.                 |
| **Azure License Type: Enterprise**                   | A végfelhasználó ügyfél vásárol az Azure-ban és a Piactéri ajánlat aláírt közvetlenül a Microsoft nagyvállalati szerződés keretében.                  |
| **Azure License Type: Vállalati viszonteladói keresztül**  | A végfelhasználó ügyfél Azure-ban és a Piactéri ajánlat lehetővé teszi a Microsoft nagyvállalati szerződés viszonteladótól vásárol.     |
| **Azure License Type: Használatalapú fizetés**                | A végfelhasználó ügyfél Azure-ban és a Piactéri ajánlat keresztül egy "Használatalapú fizetés" megállapodás, közvetlenül a Microsofttól vásárol.                |
| **Díj összege (CC)**                              | Az ügyfélnek a kell fizetnie az összeg a *ügyfél pénznem* (másolatot kap) a számlázási.                                                                 |
| **Díj összege (számítógép)**                               | Az ügyfélnek a kell fizetnie az összeg a *küldenie jóváhagyásra cége kifizetési pénznem* (számítógép).                                                                      |
| **Terhelés dátuma**                                      | Az ügyfél díj dátuma számított (általában következett közvetlenül a használati időszak).                                             |
| **Felhő-példány neve**                              | A Microsoft Cloud egy virtuális gép üzembe helyezésének lépett fel.                                   |
| **Felhő-példány neve: Azure Global**                | A nyilvános globális Microsoft cloud.                           |
| **Felhő-példány neve: Azure Government**        | Kormányzati Microsoft felhők a következő kormányok egyike: Kína, Németország, vagy az Egyesült Államok jogszabályaival.                           |
| **Gyűjtemény állapota**  | Az egy adott számolunk fel, a számlázás és a gyűjtemény életciklusának legfrissebb állapotát.  Például az állapot gyűjtése folyamatban, összegyűjteni, vagy a visszatérítés lehet.                  |
| **Core mérete**  | A virtuális gép fut az ajánlat által bemutatott virtuális magok száma.               |
| **Ügyfél városa**  | A város nevét, az ügyfél által biztosított. Ez lehet egy ügyfél az Azure-előfizetés az városa eltér.  |
| **Ügyfél-kommunikáció nyelv**        | A nyelv, az ügyfél-kommunikációhoz ajánlott.                                  |
| **Ügyfél cég neve**                  | A vállalat nevét az ügyfél által biztosított. Ez lehet egy ügyfél az Azure-előfizetés az városa eltér.                                  |
| **Vevői ország**                       | Az ügyfél által megadott ország neve.  Ez lehet egy ügyfél az Azure-előfizetés az ország eltér.                               |
| **Ügyfél pénznem (CC)**                 | Az ügyfél előnyben részesített árak és számlázás pénzneme.                            |
| **Ügyfél e-mailben**                         | A végfelhasználó ügyfél által biztosított e-mail-címe.  Ez lehet egy ügyfél az Azure-előfizetés az e-mail-cím eltér.                   |
| **Ügyfél első neve**                              | Az Utónév, az ügyfél által biztosított.  Ez lehet egy ügyfél az Azure-előfizetéshez biztosított helyadatbáziséval.                    |
| **Ügyfél utolsó neve**                               | A Vezetéknév, az ügyfél által biztosított.  Ez lehet egy ügyfél az Azure-előfizetéshez biztosított helyadatbáziséval.                    |
| **Ügyfél-azonosító**                               | Egy ügyfél hozzárendelt egyedi azonosítója.  Egy ügyfél rendelkezhet, nulla vagy több Azure piactér-beli előfizetések.                    |
| **Ügyfél fizetési típusa**                            | Az ügyfél által használt fizetési eszköz típusa.  Például ez lehet kártya, számla vagy elektronikus ellenőrzése.                             |
| **Ügyfél postai irányítószám**                             | A postai irányítószám, az ügyfél által biztosított.  Ez lehet egy ügyfél az Azure-előfizetés a megadott irányítószám eltér.                                                                 |
| **Ügyfél állapota**                                   | Az állam (cím) a vásárló által biztosított.  Ez lehet egy ügyfél az Azure-előfizetés az utolsó neve eltér.                                                                 |
| **Beszerzés dátuma**                                    | Első napjától az ügyfél az Ön által közzétett bármely ajánlatot vásárolt.                 |
| **Elveszett dátuma**                                        | A legutóbbi dátum: az ügyfél korábban megvásárolt minden ajánlat az utolsó megszakítva.                                                         |
| **Becsült kiterjesztett költség (CC)**                   | A becsült terjeszteni egy adott termékváltozat (az ügyfél pénznemben) egységek használati díját. Ez az érték nem lehet pontos kerekítési vagy csonkolási hiba miatt.                            |
| **Becsült kiterjesztett költség (számítógép)**                   | A becsült terjeszteni egy adott termékváltozat, az a dátum-használati deviza átalakítás alapján egységek használati díj kiszámítása a (a közzétevő pénznem). Ez az érték nem lehet pontos kerekítési vagy csonkolási hiba miatt.                  |
| **Becsült küldenie jóváhagyásra cége kifizetési (számítógép)**                            | A becsült befizetést egységek használat egy adott termékváltozat, deviza átalakítás azon a napon, a használati kiszámítása a (a közzétevő pénznem) alapján. Ez az érték nem lehet pontos kerekítési vagy csonkolási hiba miatt.                        |
| **Becsült díj (számítógép)**                             | Egy adott, az a dátum-használati deviza átalakítás alapuló Termékváltozat Usage egység a becsült ár kiszámítása a (a közzétevő pénznem). Ez az érték nem lehet pontos kerekítési vagy csonkolási hiba miatt.                                             |
| **Új ügyfél**                                  | Az érték értesítjük, ha egy új ügyfél használ, vagy egy megvásárlását vagy a piactér-beli több kínál először (vagy nem).  Az érték "Yes" lesz, ha a "Dátum beszerzett" azonos naptári hónap.  Az érték "No" lesz, ha a az ügyfél bármelyik az ajánlatok a naptári hónap jelentett előtt vásárolt.                                                       |
| **Termékváltozat előzetes**                                  | Az érték értesítjük, ha az "előnézet" Termékváltozattal rendelkezik címkékkel ellátott. Az érték lesz, "Igen" Ha a Termékváltozat címkével ellátott annak megfelelően, és csak az Azure-előfizetések által, üzembe helyezheti és használja ezt a képet.  Az érték "No" lesz, ha a Termékváltozat nem azonosította "előzetes".                                          |
| **Promóciós ügyfél engedélyezve van a**                   | Az érték értesítjük, ha az ügyfél proaktív módon másolásért gyártóktól származó promóciós kapcsolatba. Jelenleg azt nem bemutatót a beállítást az ügyfelek számára, így azt jelezte, hogy a tábla között a "No". Ez a funkció üzembe helyezését követően megkezdjük hogy ennek megfelelően frissíti.                                          |
| **Marketplace-en licenc típusa**                         | A Marketplace-ajánlat a számlázási mód.                             |
| **Marketplace-en licenc típusa: Számlázás az Azure-on keresztül**   | A Microsoft az ügynök ezt az ajánlatot a piactéren, és keresztül számláz az adott felhasználók az Ön nevében. (Használatalapú hitelkártya vagy vállalati számla)       |
| **Marketplace-en licenc típusa: A saját licenc használata** | A virtuális gép üzembe helyezéséhez az ügyfél által biztosított licenc kulcs szükséges. A Microsoft nem számláz ügyfelek számára a Marketplace-ajánlat, felhasználási. |
| **Marketplace-en licenc típusa: Ingyenes**                   | A Piactéri ajánlat minden felhasználó szabadon van konfigurálva. A Microsoft nem számláz ügyfelek számára a Marketplace-ajánlat, felhasználási.    |
| **Marketplace-en licenc típusa: A Microsoft a viszonteladóként**  | A Microsoft a viszonteladójával, ezt az ajánlatot a piactéren.                               |
| **Marketplace-en előfizetés cég neve**  | A vállalat nevét az ügyfél az Azure-előfizetés által biztosított.                           |
| **Marketplace-en előfizetés-azonosító**            | Az Azure-előfizetés az ügyfél a piactéren az ajánlat kifizetni társított egyedi azonosítója.  Korábban ez volt az Azure-előfizetés GUID.  Bizonyos esetekben – például egy SAAS-sorrendben nincs előfizetés nem szükséges.           |
| **A Microsoft-díj (CC)**                               | A Microsoft díj a tranzakciót az ügyfél pénznemben.                              |
| **Ajánlat neve**                                | A Marketplace kínálatának neve.                                                             |
| **Csomag típusa**                                | A Microsoft Marketplace-ajánlat típusát.  Például annak oka az lehet felügyelt alkalmazás, a virtuális gép licencek vagy a tárolón.                           |
| **Rendelés megszakítás dátuma**                         | A dátum a Marketplace-en rendelés törölve lett.                                                       |
| **Rendelés azonosítója**                                  | Az ügyfél ahhoz a Piactéri szolgáltatás egyedi azonosítója.  Virtuális gép használatalapú ajánlatok, amelyek nem tartoznak egy megrendelés.                 |
| **Rendelés vásárlás ideje**                       | A piactér sorrendben létrehozásának dátuma.                                                        |
| **Rendelési mennyiség**                       | Egy Azure-előfizetéséhez társított ugyanazon termékváltozat rendelések mennyisége.  SAAS rendeléseket azt az egy rendelés példányához tartozó helyek számát jelöli.                                            |
| **Rendelés állapota**                              | Az adatok utolsó frissítésének időpontjában a Marketplace-en rendelés állapota.  A rendelés lehet például az aktív vagy törölt.             |
| **Rendelés állapota: Aktív**                           | Az ügyfél egy megrendelés vásárolt, és nem visszavonta a rendelési.   |
| **Rendelés állapota: Meg lett szakítva**                        | Az ügyfél korábban megvásárolt egy rendelést, és ezt követően a megfelelő sorrendbe megszakítva.        |
| **Küldenie jóváhagyásra cége kifizetési összeg (számítógép)**                             | Fizetni, az az előnyben részesített küldenie jóváhagyásra cége kifizetési pénznem (PC-s).                                       |
| **Küldenie jóváhagyásra cége kifizetési pénznem (számítógép)**                           | A kifizetések használt pénznem.                                                                |
| **Küldenie jóváhagyásra cége kifizetési dátuma**                                    | A dátum a fizetési kérést küld a Microsoft a kiválasztott pénzügyi intézmény.                                                                         |
| **Küldenie jóváhagyásra cége kifizetési állapota**                                  | Az adatok utolsó frissítésének időpontjában küldenie jóváhagyásra cége kifizetési állapotát.                                   |
| **Küldenie jóváhagyásra cége kifizetési állapota: Folyamatban van**               | A tranzakció nem áll készen a küldenie jóváhagyásra cége kifizetési. (További információkért lásd a gyűjtemény állapota)                             |
| **Küldenie jóváhagyásra cége kifizetési állapota: A kifizetett**                          | A tranzakció egy korábbi küldenie jóváhagyásra cége kifizetési számítási szerepelt. Fizetős pozitív értékek, és a negatív értékek vannak szünteti fizetendő összeget ellen. |
| **Küldenie jóváhagyásra cége kifizetési állapota: Közelgő küldenie jóváhagyásra cége kifizetési**                   | A tranzakció készen áll a küldenie jóváhagyásra cége kifizetési és fog szerepelni a következő elérhető küldenie jóváhagyásra cége kifizetési számítás.                      |
| **Ár (CC)**                                   | A használat egy adott termékváltozat (az ügyfél pénznemben) egység díja.                                                                  |
| **Szolgáltató E-mail**                                 | A Microsoft és a végfelhasználó ügyfél közötti kapcsolat részt szolgáltató e-mail-címét. Ha az ügyfél vállalati viszonteladói keresztül, a viszonteladó lesz.  Ha egy Felhőszolgáltató (CSP) van szó, a kriptográfiai Szolgáltató lesz.                                                                 |
| **Szolgáltató neve**                                 | A Microsoft és a záró ügyfél közötti kapcsolat részt szolgáltató neve. Ha az ügyfél vállalati viszonteladói keresztül, a viszonteladó lesz.  Ha egy Felhőszolgáltató (CSP) van szó, a kriptográfiai Szolgáltató lesz.                                                                 |
| **Erőforrás-URI**                                  | Az elérési utat a központi telepítési összetevő volt.                                            |
| **Termékváltozat**                                           | Termékváltozat neve meghatározott közzététele során. Ajánlat előfordulhat, hogy sok termékváltozatok, de a Termékváltozat csak egyetlen ajánlatba társítva.              |
| **Termékváltozat számlázása típusa**                                 | A termékváltozat a számlázási mód.                                                                                                             |
| **Termékváltozat számlázása típusa: A saját licenc használata**         | A virtuális gép vagy szolgáltatás telepítéséhez az ügyfél által biztosított licenc kulcs szükséges. A Microsoft nem számláz ügyfelek számára a Marketplace-ajánlat, felhasználási.   |
| **Termékváltozat számlázása típusa: A Microsoft a viszonteladóként**          | A Microsoft a viszonteladójával, ez a termékváltozat.   |
| **Termékváltozat számlázása típusa: Ingyenes**                           | A Termékváltozat minden felhasználó szabadon van konfigurálva. A Microsoft nem számláz ügyfelek számára ez a Termékváltozat a használatuk.                           |
| **Termékváltozat számlázása típusa: A fizetős**                           | A Microsoft az ügynök, ez a termékváltozat és keresztül számláz az adott felhasználók az Ön nevében. (Használatalapú hitelkártya vagy vállalati számla)                   |
| **Termékváltozat számlázása típusa: Próbaverzió**                          | Az ügyfél a próbaidőszak alatt van, és konvertálja fizetni, ha nem megszakítása és törlése.                                         |
| **Tax Amount (CC)**                                  | A alkalmazni az ügyfél számlázási az ügyfél pénznem (másolatot kap) az adó összege.                                                               |
| **A tranzakció dátuma**                                 | A tranzakció dátuma a küldenie jóváhagyásra cége kifizetési reporting rögzíti.                                                                           |
| **Tranzakció típusa**                               | Typ transakce transaction, jelentett folyamatban van.  Például ez lehet egy ingyenesen, visszatérítés vagy küldenie jóváhagyásra cége kifizetési illesztését.                                                                                        |
| **Próbaverziós befejezési dátum**                                 | A dátum a próbaidőszak alatt a megrendelés megszűnik, vagy véget ért.                                                                           |
| **Használat dátuma**                                       | A dátum-ügyfelek általi használatot történt.                                                                                                             |
| **Használati mennyiséget**                                       | A Termékváltozat jelentett ügyfél használatát. A Virtuálisgép-lemezképek esetében rekordok jelentett az időszakra, a virtuális gép mérete és a Termékváltozat, a használati képviseli.                                               |
| **Használati útmutató**                                       | Egy vagy több napot az ügyfelek általi használatot a társított bejegyzés az küldenie jóváhagyásra cége kifizetési jelentés az adott Termékváltozat azonosítója.                                               |
| **Használat típusa**                                       | A használati mért leírása. (Normalizált használati vagy nyers használat)                                                              |
| **Használat típusa: Normalizált használat**                | Részt vesz a használati Virtuálisgép-magok számát a fiók használati órák normalizálva: [számát a virtuális gép magjainak] x [óra a nyers Usage]. A virtuális gépek szerint "SHAREDCORE" 1/6 (vagy 0.1666) használja a [számát a virtuális gép magjainak] szorzójaként.                                        |
| **Használat típusa: Nyers használat**                        | A piactéren az ajánlat használati órán keresztül.  "Naptár" óra tekintetében jelennek meg.  Idő a virtuális gépek száma előzetesen működniük.                           |
| **Használati egység**                                   | A rögzített használat mérési egysége. Virtuális gépek mindig az óránkénti mértékegységek mérése történik.                               |
| **VM Size**                                       | A virtuális gép hardver mérete az Azure-ajánlatát igazítva jelöli. Ilyenek például `Basic_A0`, `Standard_A11`, `Standard_D12`, és `Standard_G4`.   |
