# <a name="how-do-azure-active-directory-admins-add-b2b-collaboration-users"></a>Hogyan adnak hozzá az Azure Active Directory-rendszergazdák vállalatközi együttműködési felhasználókat?

Ahogy az alábbi videó az B2B együttműködés felhasználók hozzáadása az Azure AD a rendszergazdák csoporthoz:
