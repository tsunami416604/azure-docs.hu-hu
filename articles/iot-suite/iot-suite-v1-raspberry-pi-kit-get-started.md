# <a name="connect-your-microsoft-azure-iot-raspberry-pi-3-starter-kit-to-the-remote-monitoring-solution"></a>A Microsoft Azure IoT málna Pi 3 Starter Kit kapcsolódni a távoli felügyeleti megoldás

Az oktatóanyagok ebben a szakaszban megtudhatja, hogyan málna Pi 3 eszköz csatlakozni a távoli felügyeleti megoldást nyújt segítséget. Válassza ki az előnyben részesített programozási nyelv megfelelő oktatóanyag és a hogy van-e a használhatók a málna Pi a hardverét.

## <a name="the-tutorials"></a>Az oktatóanyagok

| Oktatóanyag | Megjegyzések | Nyelvek |
| -------- | ----- | --------- |
| Szimulált telemetriai (alapszintű)| Érzékelőadatok szimulálja. Önálló málna Pi használja. | [C][lnk-c-simulator], [Node.js][lnk-node-simulator] |
| Valós érzékelő (közepes) | Csatlakozik a málna Pi BME280 érzékelő adatokat használ. | [C][lnk-c-basic], [Node.js][lnk-node-basic] |
| Valósítja meg a belső vezérlőprogram-frissítés (speciális)| Csatlakozik a málna Pi BME280 érzékelő adatokat használ. Lehetővé teszi, hogy a málna Pi távoli belső vezérlőprogram-frissítések. | [C][lnk-c-advanced], [Node.js][lnk-node-advanced] |
