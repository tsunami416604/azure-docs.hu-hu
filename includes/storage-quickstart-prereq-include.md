Hozzáférhet az Azure Storage, Azure-előfizetésre lesz szüksége. Ha még nincs előfizetése, hozzon létre egy [ingyenes fiókot](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) megkezdése előtt.

Minden, az Azure Storage-hozzáférés tárfiókon keresztül történik. Ez a rövid útmutatóhoz hozzon létre egy tárolási fiókot a [az Azure portal](https://portal.azure.com/), az Azure PowerShell vagy Azure parancssori felület. A fiók létrehozásával kapcsolatos útmutatóért lásd a [tárfiók létrehozását](../articles/storage/common/storage-quickstart-create-account.md) ismertető szakaszt.
