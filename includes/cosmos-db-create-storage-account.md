Az Azure Storage-fiók létrehozásának legegyszerűbb módja a [Azure Portal](https://portal.azure.com)használata. További tudnivalókért lásd: [Create a storage account](../articles/storage/common/storage-account-create.md) (Tárfiók létrehozása).

Ezenkívül az [Azure PowerShell](/powershell/module/az.storage/) vagy az [Azure CLI](../articles/storage/blobs/storage-quickstart-blobs-cli.md) használatával is létrehozhat egy Azure-tárfiókot.

Ha jelenleg nem kívánja létrehozni a Storage-fiókot, az Azure Storage Emulator használatával futtathatja és tesztelheti a programkódot egy helyi környezetben. További információ: [Az Azure Storage Emulator használata fejlesztéshez és teszteléshez](../articles/storage/common/storage-use-emulator.md).
